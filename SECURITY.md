# Security Policy

## Supported Versions

This repository is a v0.1 research preview. It is not production-ready and does not provide security support for production deployments.

## Reporting a Vulnerability

Report security issues through GitHub Security Advisories, or email `dev404ai@gmail.com` as a fallback.

Please include:

- affected file or workflow;
- reproduction steps;
- expected and observed behavior;
- whether the issue affects local verification only or could expose user data.

Do not report unsupported production-use issues as vulnerabilities; the repository is an inspectable reference implementation, not a production platform.
